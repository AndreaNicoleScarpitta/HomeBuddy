Overview

Home Buddy is a comprehensive home maintenance assistant web application that helps homeowners track maintenance tasks, manage budgets, and get AI-powered guidance for home care decisions.

Key Value Proposition: Combines task management, budget tracking, and AI assistance into a single platform designed with an empathetic, no-shame approach to home ownership.

Technology Stack

Frontend
React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui, Wouter routing
Backend
Express.js, TypeScript, Node.js, Pino logging
Database
PostgreSQL with Drizzle ORM
Authentication
Replit Auth (OpenID Connect)
AI Integration
OpenAI GPT-4o with streaming responses
File Storage
Replit Object Storage for document uploads

Design System

ElementValue

Features

Home Profile
Address management with optional USPS verification and Google Places autocomplete
Maintenance Tasks
Track tasks with urgency levels (now/soon/later), DIY ratings, and cost estimates
AI Assistant
Streaming chat with GPT-4o for DIY guidance, safety warnings, and recommendations
Budget Tracker
Funds with allocations and expense tracking (amounts stored in cents)
Inspection Reports
Upload PDF/images for analysis (currently mock AI - sample findings generated)
PWA Support
Installable progressive web app with manifest and service worker
Note: Inspection report analysis currently uses mock data. Real AI analysis can be implemented by integrating document parsing with OpenAI Vision API.

API Endpoints

All endpoints except /api/contact and /api/health require authentication via Replit Auth.

Authentication

GET/api/auth/userGet current user
GET/api/logoutLogout user

Home Profile

GET/api/homeGet user's home
POST/api/homeCreate home profile
PATCH/api/home/:idUpdate home

Address Verification

POST/api/address/verifyVerify address via USPS
GET/api/address/statusCheck if USPS is configured

Systems

GET/api/home/:homeId/systemsList home systems
POST/api/home/:homeId/systemsCreate system

Maintenance Tasks

GET/api/home/:homeId/tasksList tasks
POST/api/home/:homeId/tasksCreate task
PATCH/api/tasks/:idUpdate task
DELETE/api/tasks/:idDelete task

AI Chat

GET/api/home/:homeId/chatGet chat history
POST/api/home/:homeId/chatSave message (streaming via SSE)

Funds

GET/api/home/:homeId/fundsList funds
POST/api/home/:homeId/fundsCreate fund
PATCH/api/funds/:idUpdate fund
DELETE/api/funds/:idDelete fund

Allocations

GET/api/funds/:fundId/allocationsList allocations by fund
GET/api/tasks/:taskId/allocationsList allocations by task
POST/api/allocationsCreate allocation
PATCH/api/allocations/:idUpdate allocation
DELETE/api/allocations/:idDelete allocation

Expenses

GET/api/home/:homeId/expensesList all expenses
GET/api/funds/:fundId/expensesList expenses by fund
POST/api/expensesCreate expense
PATCH/api/expenses/:idUpdate expense
DELETE/api/expenses/:idDelete expense

Inspection Reports

GET/api/home/:homeId/reportsList reports
GET/api/reports/:idGet report with findings
POST/api/home/:homeId/reportsCreate report
POST/api/reports/:id/analyzeTrigger analysis (mock)
DELETE/api/reports/:idDelete report

Other

POST/api/contactSubmit contact form (public)
GET/api/healthHealth check endpoint
