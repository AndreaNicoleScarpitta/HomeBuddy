You are a senior Staff Product Designer + Security-aware Full Stack Engineer working inside an existing production app called **Home Buddy**.

Your task is to INSERT a new feature called **Circuit Panel Mapping** into the app while maintaining strict alignment with:

* Existing UX workflow structure
* Established design language and navigation patterns
* Historical privacy and security architecture
* Staff-designer principles: clarity, minimal friction, progressive disclosure, and system-first modeling

You must EXTEND the system, not redesign it.

---

# CONTEXT

Home Buddy already has:

* Dashboard (Home Base)
* Systems section (HVAC, Plumbing, Electrical, etc.)
* AI Assistant with photo analysis
* Inspections upload
* Documents storage
* Maintenance History
* Profile/Settings

Key principles already established in the app:

1. The Dashboard is the operational hub.
2. Systems represent the “source of truth” for structural home data.
3. AI features assist but never become the primary configuration UI.
4. UX tone is calm, professional, and anxiety-aware.
5. Security prioritizes user ownership, consent, and minimal data retention.

DO NOT introduce new global navigation items.
DO NOT move existing features.
DO NOT alter onboarding or authentication.

# OBJECTIVE

Add a new capability allowing users to:

* Take a photo of their home’s electrical breaker panel
* Annotate breakers with room/system labels
* Save the mapping as structured system data
* Use the mapping later for AI guidance and maintenance workflows

This must feel like a natural extension of the Electrical system.

Follow existing UX patterns already present in the app:

# SECURITY & PRIVACY REQUIREMENTS

You MUST align with historical security posture:

DATA HANDLING:

* Images uploaded via presigned URL (GCS).
* No direct client → server binary payloads.
* Metadata stored separately from image asset.

CONSENT:

* Reuse existing Vision API consent modal.
* Add a checkbox: “Store panel image for future reference.”
* If unchecked, only annotations persist.

MINIMAL STORAGE:

* Store breaker annotations as structured JSON.
* Never store EXIF location metadata.
* Strip background metadata before upload.

ACCESS CONTROL:

* Circuit Map is scoped to the authenticated user’s home profile.
* Respect existing OAuth session model.
* No cross-home visibility.

AI SAFETY:

* AI suggestions are optional overlays.
* User remains final authority over labels.
